
  # Tenant Management Platform UI

  This is a code bundle for Tenant Management Platform UI. The original project is available at https://www.figma.com/design/jx7KuVM0UWwFEZiH1oj5Te/Tenant-Management-Platform-UI.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Testing

  Tests use Vitest with React Testing Library.

  Run `npm run test` to execute the test suite.
